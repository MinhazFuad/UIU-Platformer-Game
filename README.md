
# University Platformer Game

A 2D platformer game built with **JavaFX**, designed to help new university students navigate campus life in an engaging and interactive way.

---

## Overview

**University Platformer Game** combines adventure, learning, and problem-solving into a single interactive experience. Players explore campus-inspired levels, encounter real-life student scenarios, and engage with puzzle-based challenges that enhance critical thinking.

---

## Features

* **Immersive 2D Gameplay:** Smooth controls and platform mechanics developed using JavaFX.
* **Campus-Inspired Levels:** Each level reflects aspects of university life, from orientation to exams.
* **Puzzle Challenges:** Embedded mini-games that promote logical reasoning and problem-solving skills.
* **Scoring and Achievements:** Players earn points, unlock new levels, and gain insights into campus activities.
* **Fresher-Friendly Experience:** Designed to help new students adapt and engage with university life.

---

## Tech Stack

* **Language:** Java
* **Framework:** JavaFX
* **Tools:** Scene Builder, IntelliJ IDEA / Eclipse
* **Graphics:** Custom sprites and level assets

---

## Getting Started

### Prerequisites

* Java 17 or later
* JavaFX SDK
* IDE such as IntelliJ IDEA or Eclipse

### Clone the Repository

```bash
git clone https://github.com/your-username/UIU-Platformer-Game.git
cd UIU-Platformer-Game
```

## Gameplay

Players take on the role of a university fresher navigating through various levels inspired by campus life. Each stage presents new challenges, puzzles, and achievements to unlock, providing an engaging and educational experience.
